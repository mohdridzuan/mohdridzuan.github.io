Para 2 : What is CSS?

  • CSS stands for Cascading Style Sheet
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

Para 3 : CSS can be written in 2 approach
Approach For Modifying Attributes
Approach Description
Using STYLE Defining the attribute for each tag locally (inline>
Using CSS
  • Written in the head section of the HTML
  • The CSS is applicable to a specific tags, ID or CLASS
